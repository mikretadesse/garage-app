# Abe's Garage App

[![Frontend](https://img.shields.io/badge/Frontend-React-blue)]()  
[![Backend](https://img.shields.io/badge/Backend-Node.js-green)]()  

Abe's Garage App is a **full-stack garage management application** designed to streamline and enhance the day-to-day workflow of Abe's Garage. The application enables employees to manage customers, orders, and tasks efficiently while providing customers the ability to track order status remotely. The system emphasizes security, scalability, and maintainability.
